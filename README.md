# Weather App

A Flutter application that displays real-time weather information using a weather API with a clean and responsive user interface.

## Features

* Search weather by city
* Display current temperature
* Show weather conditions
* Clean and responsive UI

## Technologies

* Flutter
* Dart
* REST API
* HTTP

## Getting Started

```bash
flutter pub get
flutter run
```
